1) Push this whole folder to GitHub with GitHub Desktop.
2) Deploy scanner with main path: scanner_pro/app.py
3) After first push, open GitHub -> Actions -> run workflow 'daily-refresh' once.
4) The app is snapshot-only. It becomes current after the workflow/local build writes fresh universes and scanner snapshots.
5) If you want a local full rebuild instead: run install_and_build.bat or python build_everything_full.py
